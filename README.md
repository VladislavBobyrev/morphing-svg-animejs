 ## Для запуска потребуется

- Этот репозиторий 
 ```
git clone https://github.com/VladislavBobyrev/morphing-svg-animejs
```
- Node.js 
 
```
https://nodejs.org/en/
```

- Браузер

## Команды

|                        |                       |
|------------------------|:----------------------|
| Установить зависимости | `npm i`               |
| Запустить локально     | `npm run serve`       |
| Собрать                | `npm run build`       |
